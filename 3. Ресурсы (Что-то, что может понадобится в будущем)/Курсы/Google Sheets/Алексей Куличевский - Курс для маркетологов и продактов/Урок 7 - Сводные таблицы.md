#### Что такое сводные таблицы
Это вид таблиц, который позволяет удобно анализировать данные. Сделать таблицу сводной можно через Insert -> Pivot table. У любой сводной таблицы есть 4 рычага: строки, колонки, значения и фильтры. Больше описания не будет. Только практика. 
Удобно агрегировать сырые данные (каждая строка - отдельный пользователь).
