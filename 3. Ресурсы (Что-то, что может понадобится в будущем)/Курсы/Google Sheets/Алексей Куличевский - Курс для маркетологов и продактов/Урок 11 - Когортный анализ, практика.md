#### Алгоритм
0. Понимаем задачу, которую надо решить. 
1. Определяем когорту (событие, промежуток времени). 
2. Определяем целевые метрики.
3. Строим календарный отчет, чтобы посмотреть распределение метрики по когортам.
4. Строим возрастной отчет, чтобы сравнить динамику развития когорт между собой.
#### План урока и сбора таблицы непосредственно
1. Смотрим на сырые данные и собираем дату первой сессии с помощью сводной таблицы. Создаем новую колонку - first_session и помещаем в нее информацию из сводной таблицы с помощью VLOOKUP. Затем собираем сводную таблицу (календарный отчет), где строки это дата первой сессии, колонки - дата сессии, а значения - COUNTUNIQUE первой сессии.
2. Теперь надо собрать возраст когорты для того чтобы посмотреть на возрастной отчет. Для этого надо из даты сессии вычесть дату первой сессии (или DATEDIF(первая сессия, дата сессии, "D")). Теперь можем почитать возрастной отчет в котором: строки - первая сессия, колонки - возраст когорты, а значения - процент по строке уникальных значений user_id.
